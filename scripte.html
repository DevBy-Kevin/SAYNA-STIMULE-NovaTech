Calculateur de Devis

Script de calcul automatique de devis pour projets digitaux

Ce script permet d'estimer le coût d'un projet digital en fonction de ses caractéristiques : type de projet, complexité, design, fonctionnalités, etc.


#!/usr/bin/env python3
# -*- coding: utf-8 -*-

"""
Calculateur de devis pour NovaTech
Ce script calcule le prix d'un projet digital en fonction de ses caractéristiques
"""

def calculer_prix_projet(type_projet, nb_pages, design, fonctionnalites, seo):
    # Prix de base selon le type de projet
    prix_base = {
        'website': 1500,
        'ecommerce': 3500,
        'webapp': 5000,
        'mobile': 8500,
        'desktop': 7000
    }
    
    # Calcul du prix total
    total = prix_base.get(type_projet, 0)
    total += nb_pages * 100  # 100€ par page
    total += (design - 1) * 500  # Supplément design
    total += sum(fonctionnalites)  # Fonctionnalités supplémentaires
    total += seo  # Option SEO
    
    # Application des remises pour les gros projets
    if total > 10000:
        total *= 0.9  # 10% de remise
    elif total > 5000:
        total *= 0.95  # 5% de remise
    
    return round(total, 2)

def afficher_devis(type_projet, nb_pages, design, fonctionnalites, seo, total):
    # Affichage formaté du devis
    print(f"\n=== DEVIS NOVATECH ===")
    print(f"Type de projet: {type_projet}")
    print(f"Nombre de pages: {nb_pages}")
    print(f"Niveau de design: {design}")
    print(f"Fonctionnalités: {len(fonctionnalites)} option(s)")
    print(f"SEO: {seo}€")
    print(f"-----------------------")
    print(f"TOTAL: {total}€")
    print(f"=======================\n")

if __name__ == "__main__":
    # Exemple d'utilisation
    type_projet = "website"
    nb_pages = 5
    design = 1.5  # 1 = Basique, 1.5 = Personnalisé, 2 = Premium
    fonctionnalites = [500, 800]  # Formulaire de contact + Espace membre
    seo = 300  # SEO Basique
    
    total = calculer_prix_projet(type_projet, nb_pages, design, fonctionnalites, seo)
    afficher_devis(type_projet, nb_pages, design, fonctionnalites, seo, total)

Explications

Ce script utilise une approche modulaire pour calculer le prix d'un projet digital :

  • Prix de base selon le type de projet
  • Supplément par page (100€ par page)
  • Niveau de design (basique, personnalisé, premium)
  • Fonctionnalités supplémentaires avec coûts variables
  • Options SEO avec différents niveaux de tarification
  • Remises progressives pour les gros projets

La fonction afficher_devis() formatte les résultats pour une présentation cliente professionnelle.

Générateur de Mots de Passe

Outil de création de mots de passe sécurisés

Ce script génère des mots de passe forts et sécurisés avec différentes options de complexité pour protéger les accès aux projets de nos clients.


#!/usr/bin/env python3
# -*- coding: utf-8 -*-

"""
Générateur de mots de passe sécurisés pour NovaTech
"""

import random
import string

def generer_mot_de_passe(longueur=12, majuscules=True, chiffres=True, symboles=True):
    # Caractères possibles
    caracteres = string.ascii_lowercase  # Lettres minuscules
    
    if majuscules:
        caracteres += string.ascii_uppercase  # Lettres majuscules
    
    if chiffres:
        caracteres += string.digits  # Chiffres
    
    if symboles:
        caracteres += string.punctuation  # Symboles
    
    # Vérification de la longueur minimale
    if longueur < 8:
        print("Attention : Un mot de passe de moins de 8 caractères n'est pas sécurisé !")
        longueur = 8
    
    # Génération du mot de passe
    mot_de_passe = ''.join(random.choice(caracteres) for i in range(longueur))
    
    return mot_de_passe

def evaluer_force(mot_de_passe):
    # Évaluation de la force du mot de passe
    score = 0
    
    if len(mot_de_passe) >= 12:
        score += 2
    elif len(mot_de_passe) >= 8:
        score += 1
    
    if any(c in string.digits for c in mot_de_passe):
        score += 1
    
    if any(c in string.punctuation for c in mot_de_passe):
        score += 1
    
    if any(c in string.ascii_uppercase for c in mot_de_passe):
        score += 1
    
    if score == 5:
        return "Très fort"
    elif score >= 3:
        return "Fort"
    elif score >= 2:
        return "Moyen"
    else:
        return "Faible"

if __name__ == "__main__":
    # Exemple d'utilisation
    print("=== GÉNÉRATEUR DE MOTS DE PASSE NOVATECH ===")
    
    # Génération de 3 mots de passe avec différentes complexités
    for i in range(3):
        mdp = generer_mot_de_passe(longueur=12, majuscules=True, chiffres=True, symboles=True)
        force = evaluer_force(mdp)
        print(f"Mot de passe {i+1}: {mdp} (Force: {force})")

Explications

Ce script génère des mots de passe sécurisés avec plusieurs fonctionnalités :

  • Personnalisation de la longueur (8 caractères minimum)
  • Inclusion optionnelle de majuscules, chiffres et symboles
  • Évaluation de la force du mot de passe généré
  • Génération aléatoire sécurisée avec le module random

L'algorithme évalue la force du mot de passe en fonction de :

  • Longueur du mot de passe
  • Présence de chiffres
  • Présence de symboles
  • Présence de majuscules

Analyseur de Sentiments

Outil d'analyse des sentiments des témoignages clients

Ce script analyse les témoignages clients pour déterminer leur sentiment (positif, négatif ou neutre) et aide à améliorer notre service client.


#!/usr/bin/env python3
# -*- coding: utf-8 -*-

"""
Analyseur de sentiments pour NovaTech
Ce script analyse les témoignages clients pour déterminer leur ton (positif, négatif, neutre)
"""

import re

class AnalyseurSentiments:
    def __init__(self):
        # Mots-clés positifs et négatifs (simplifié pour l'exemple)
        self.mots_positifs = {
            'excellent': 2, 'super': 2, 'génial': 2, 'parfait': 2,
            'bon': 1, 'agréable': 1, 'satisfait': 1, 'rapide': 1,
            'efficace': 1, 'professionnel': 1, 'qualité': 1, 'recommande': 2
        }
        
        self.mots_negatifs = {
            'mauvais': 2, 'horrible': 2, 'nul': 2, 'déçu': 2,
            'lent': 1, 'cher': 1, 'compliqué': 1, 'problème': 1,
            'bug': 1, 'erreur': 1, 'insatisfait': 2, 'déception': 2
        }

    def analyser_texte(self, texte):
        # Nettoyage du texte
        texte = texte.lower()
        texte = re.sub(r'[^\w\s]', '', texte)  # Supprimer la ponctuation
        
        mots = texte.split()
        
        # Calcul des scores
        score_positif = 0
        score_negatif = 0
        
        for mot in mots:
            if mot in self.mots_positifs:
                score_positif += self.mots_positifs[mot]
            if mot in self.mots_negatifs:
                score_negatif += self.mots_negatifs[mot]
        
        # Détermination du sentiment
        score_total = score_positif + score_negatif
        
        if score_total == 0:
            sentiment = "neutre"
            confiance = 0
        else:
            ratio = score_positif / score_total if score_total > 0 else 0
            
            if ratio > 0.6:
                sentiment = "positif"
            elif ratio < 0.4:
                sentiment = "négatif"
            else:
                sentiment = "neutre"
            
            confiance = min(100, abs(score_positif - score_negatif) * 10)
        
        return {
            "sentiment": sentiment,
            "confiance": confiance,
            "score_positif": score_positif,
            "score_negatif": score_negatif
        }

def afficher_resultat(texte, resultat):
    print(f"\n=== ANALYSE DE SENTIMENT ===")
    print(f"Texte: \"{texte}\"")
    print(f"Sentiment: {resultat['sentiment']}")
    print(f"Confiance: {resultat['confiance']}%")
    print(f"Score positif: {resultat['score_positif']}")
    print(f"Score négatif: {resultat['score_negatif']}")
    print(f"============================\n")

if __name__ == "__main__":
    # Exemple d'utilisation
    analyseur = AnalyseurSentiments()
    
    # Textes de test
    textes = [
        "Service excellent et équipe très professionnelle. Je recommande !",
        "Déçu par la lenteur de réponse et les bugs rencontrés.",
        "Le produit est correct mais un peu cher pour ce que c'est."
    ]
    
    for texte in textes:
        resultat = analyseur.analyser_texte(texte)
        afficher_resultat(texte, resultat)

Explications

Ce script analyse le sentiment d'un texte en utilisant une approche par dictionnaire de mots-clés :

  • Dictionnaires de mots positifs et négatifs avec pondération
  • Nettoyage du texte (minuscules, suppression de la ponctuation)
  • Calcul de scores basé sur la présence des mots-clés
  • Détermination du sentiment (positif, négatif, neutre)
  • Niveau de confiance basé sur l'écart entre scores

Les applications de ce script incluent :

  • Analyse des témoignages clients
  • Surveillance de la réputation en ligne
  • Mesure de la satisfaction client
  • Identification des points d'amélioration
