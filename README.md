### 🔐 Générateur de Mots de Passe

```bash
python generateur_mdp.py -l 16 -n 3
```

Options disponibles :

* `-l N` → Longueur (8-64)
* `-n N` → Nombre de mots de passe
* `-s` → Exclure les symboles
* `-q` → Mode silencieux
